# Standards

Human-facing summary of the standards applied to Werkbank. The agent contract lives in `AGENTS.md`.

## Applied

- **kebab-case ASCII-only paths** — lowercase, hyphens, no spaces or underscores.
- **English structural paths** — directory and file names in English. Document content may use Dutch where natural.
- **ISO 8601 dates** — `YYYY-MM-DD` prefixes on date-bearing filenames.
- **Conventional Commits 1.0.0** — `<type>(<scope>): <description>`.
- **Keep a Changelog 1.1.0** — version sections in `CHANGELOG.md`.
- **IEEE 830-style requirements** — the design spec uses observable, testable requirements.

## Not applied (out of scope for a personal study site)

- ISO/IEC/IEEE 42010 architecture views — overkill for a single-page static site.
- ISO 29119-3 test documentation — covered by inline test files, not formal reports.
